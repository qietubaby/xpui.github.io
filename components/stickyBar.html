    # StickyBar 固钉条

    # 使用方法

    • 请滑动滚动条才能看到效果
    • 固钉条总是会固定在页面中的某个元素之上，如果该元素因为滚动条隐藏，那么会显示在底部
    • 优先检查用户是否传入stickyName

    # 案例

    footer元素
    固钉条默认总是悬浮此在元素之上

    # Props

    参数 说明 类型 可选值 默认值
    stickyElement sticky悬浮在哪个元素之上，传入id时请记得加# string - -
    distace sticky距离底部的距离，如果传入了stickyElement将失效 number - 0
